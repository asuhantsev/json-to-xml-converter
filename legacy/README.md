# Legacy

This folder stores historical or non-core artifacts moved out of the root repo to keep the project clean.

## Contents
- `frontend/` - old unrelated frontend snapshot.
- `ChatExport_2025-02-09/` - historical Telegram export archive.
- `sources/` - legacy sample files.
- `exports/` - large historical XML outputs moved from root.

Core runtime files remain in the root (`jsontoxml.py`, `src/`, `scripts/`, `tests/`, `docs/`).
